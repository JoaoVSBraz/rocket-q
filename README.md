## 📚 Sobre
---
<p>
Rocket.Q é uma plataforma que cadastra perguntas anônimas. Ideal para eventos online como streams.
</p>
<p>
O projeto foi desenvolvido durante o evento online Next Level Week Together (#NLW6) pela Rocketseat.
